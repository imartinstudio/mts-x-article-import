🔒 Privacy Policy · 隐私政策

X Article Import — Chrome Extension

Last updated: 2026-07-04

1. Data Collection · 数据收集

X Article Import does NOT collect, store, or transmit your article content to any server.

This extension operates entirely within your browser. It does not:

  • ❌ Upload your Markdown files or media to external servers
  • ❌ Collect your X/Twitter credentials or authentication tokens
  • ❌ Track your browsing history across other websites
  • ❌ Send analytics, telemetry, or crash reports to third parties

2. How the Extension Works · 扩展工作原理

All processing happens locally on your device:

  • The extension activates on https://x.com/*, primarily the X Articles editor (/compose/articles).
  • You choose a local Markdown file and optional media files through the browser file picker. Files are read in-memory and never leave your device.
  • Parsed content is written into the X Articles draft editor through DOM automation and a page-context script — the same surface you would edit manually.
  • Optional tables or Mermaid diagrams may be converted to images locally (Canvas) before insertion. No network request is made for that conversion.

3. Local Storage · 本地存储

The extension may persist a single user preference locally:

  • Subscription tier (X Premium vs Premium+) — stored in chrome.storage.local with a localStorage fallback on the X page. Used only to choose formatting rules during import.

No article text, file paths, or media are stored beyond the active import session.

4. Permissions Explained · 权限说明

Permission Why it's needed
storage Save your X Premium / Premium+ preference between sessions.
scripting Inject the draft-writer helper into the X Articles page context when importing.
clipboardRead / clipboardWrite Support clipboard-related editor interactions required by the X Articles surface.
host_permissions (x.com) Required by Chrome Manifest V3 to run on X. The extension does not run on other sites.

5. Third-Party Services · 第三方服务

The extension bundle includes open-source libraries (e.g. Mermaid for diagram rendering) that execute locally. They do not phone home. The only network traffic is your normal use of x.com itself.

6. Children's Privacy · 儿童隐私

This extension does not knowingly collect information from children under 13.

7. Changes to This Policy · 政策更新

If this privacy policy changes, the updated version will be posted at the same URL. The "Last updated" date at the top will reflect the revision date.
