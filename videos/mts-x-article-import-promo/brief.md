# Prompt Expansion

## Intent
制作一支关于 `mts-x-article-import` Chrome 扩展的中文宣传推广视频：从安装构建、加载扩展，到在 X Articles 中选择 Markdown、预览适配、本地媒体匹配、导入完成，形成完整闭环。

## Audience
面向写长文、使用 Markdown 管理草稿、需要把内容发布到 X Articles 的创作者和开发者。

## Narrative Arc
1. 痛点：Markdown 到 X Articles 不再复制粘贴。
2. 安装：开源项目，本地构建，Chrome 加载 `dist/`。
3. 导入：打开 `x.com/compose/articles`，点击「导入 Markdown」。
4. 适配：封面、图片、代码块、Premium/Premium+ 规则、本地媒体匹配。
5. 结果：完整文章写入编辑器，纯本地处理，准备发布。

## Rhythm
fast-fast-SLOW-fast-SHADER-hold：开场快速建立价值；安装与导入用中速演示；预览适配给足停留；完成场景用强 CTA 收束。

## Composition
1920x1080，约 38 秒，单个 `index.html` 根合成，录屏片段作为独立 video clips，文字和 UI 注释作为 overlay clips。
